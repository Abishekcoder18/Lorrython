Design a modern full-stack web application UI for a logistics AI platform called **“Lane Intelligence Builder Agent.”**

The platform automatically cleans messy shipment data, standardizes logistics fields, clusters shipments into transportation lanes, and generates RFQ-ready datasets for procurement teams.

Design a **clean enterprise dashboard interface** suitable for supply chain analysts and logistics managers.

Use a **professional tech style** with:

* Color palette: dark blue (#0B3C5D), teal (#1ABC9C), light grey (#F5F7FA), white
* Minimal modern layout
* Rounded cards
* Clear data tables
* Analytics dashboards
* Simple icons for logistics and AI

Create the following **main screens**:

1. **Login Page**

   * Title: Lane Intelligence Builder
   * Email field
   * Password field
   * Login button
   * Illustration showing logistics data and AI analytics
   * Small text: “AI-Powered Logistics Data Intelligence”

2. **Main Dashboard**

   * Left sidebar navigation
   * Top header with search bar and user profile
   * Dashboard cards showing:

     * Total Shipments Processed
     * Cleaned Records
     * Detected Lanes
     * Duplicate Records Removed
   * Graphs:

     * Shipment volume trends
     * Lane distribution
     * Carrier usage analytics
   * Recent activity panel

3. **Data Upload Page**

   * Upload shipment dataset (CSV/Excel)
   * Drag-and-drop upload box
   * Dataset preview table
   * Button: “Run Cleaning Engine”

4. **Data Cleaning Engine Page**

   * Progress status visualization
   * Panels showing:

     * Schema detection
     * Location normalization
     * Equipment standardization
     * Deduplication results
   * Before vs After data comparison table

5. **Lane Clustering Page**

   * Visualization of shipment lanes
   * Map with origin-destination connections
   * Table listing:

     * Lane ID
     * Origin
     * Destination
     * Shipment Volume
     * Average Freight Rate
   * Button: “Generate Lane Insights”

6. **RFQ Dataset Generator**

   * Table of cleaned shipment lanes
   * Filters for carrier, service level, equipment
   * Export buttons:

     * Download CSV
     * Generate RFQ dataset

7. **Data Quality Analytics Page**

   * Charts showing:

     * Data inconsistency detection
     * Duplicate shipment detection
     * Location normalization results
   * AI insights panel

8. **Settings Page**

   * Configure data cleaning rules
   * Location normalization dictionary
   * Equipment type mapping
   * User account settings

Design reusable **components**:

* Sidebar navigation
* Data tables
* Analytics cards
* Chart components
* Upload widgets
* Status indicators

Use icons related to:

* logistics
* trucks
* maps
* AI analytics
* data processing

Ensure the design looks like a **modern SaaS logistics analytics platform** similar to supply chain intelligence tools used by enterprise companies.
